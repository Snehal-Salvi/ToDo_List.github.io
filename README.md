# TO-DO LIST

To-Do List project helps us to create our To-Do tasks.

## Features

- Light/dark mode toggle
- Fullscreen mode
- Add new task
- Delete task
- Mark completed tasks
- Track the count of tasks left
- See completed uncompleted tasks

## Technlogies Used

- HTML
- CSS
- Javascript
